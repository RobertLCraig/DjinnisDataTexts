# Release Notes

## Version: 0.1.0

### Initial Release

- Core scaffold with DDT namespace, module registration system, and DGF migration logic
- Ported Guild, Friends, and Communities modules from DjinnisGuildFriends (DDT- prefix)
- New Spec Switch module: talent/loadout/loot spec switching with clickable Djinni tooltip
- New Saved Instances module: raid/dungeon lockout summary with boss details and alt integration
- DemoMode support for development outside the game client
- Blizzard Settings API integration with per-module subcategories
- Deploy and release scripts for Windows
